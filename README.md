# Clip Bookmark Chrome Extension (Manifest V3)

Twitchの配信またはVOD（アーカイブ）の視聴中に、経過時間や再生位置を簡単かつ安全に打刻・保存し、後から確認や該当時間へジャンプができる軽量・安全設計のGoogle Chrome拡張機能です。

---

## 🌟 主な特徴

1. **極限の低負荷設計 (パフォーマンス重視)**
   - 設定でチャット監視をOFFにした場合、内部の `MutationObserver` の接続を完全に切断（`disconnect`）し、不要な監視処理によるブラウザのCPU負荷を **0%** に抑えます。
   - ポップアップUIは Vanilla TypeScript と `DocumentFragment` を用いて、履歴が100件以上存在する場合でも **0.2秒以下** の高速描画を実現。
2. **多層セキュリティ防御 (Security & Safety)**
   - **XSS対策**: DOMから取得したタイトルやチャンネル名の挿入時は、HTMLエスケープと制御文字の除去、最大255文字への切り捨て（サニライズ）を徹底し、`textContent` による安全な描画を行います。
   - **送信元検証**: バックグラウンド Service Worker でメッセージを受信する際、送信元の拡張機能ID、Twitchドメイン（`twitch.tv` / `*.twitch.tv`）、およびタブオブジェクトの存在を検証し、外部からのメッセージインジェクションを防ぎます。
   - **遷移先検証**: ブックマークからの遷移時は、正規表現パターンによる厳格な検証を通った安全なTwitchのURLのみを `chrome.tabs.create` に渡します。
3. **FIFOストレージローテーション**
   - 保存されたブックマーク履歴が 1,000件 に達した時点で、最も古いものから順に自動削除（FIFO方式）を行い、ブラウザのストレージ容量を圧迫しないよう制限します。
4. **フルスクリーン対応トースト通知**
   - 打刻時に表示されるトースト通知は、Twitchのビデオプレイヤーコンテナ（`.video-player__container`）内にマウントされるため、HTML5フルスクリーン表示中も映像の最前面に正しく表示されます。
   - トースト自体はマウス透過（`pointer-events: none`）としつつ、閉じるボタンのみクリック可能（`pointer-events: auto`）なハイブリッド構成です。

## 📂 プロジェクト構成

```text
src/
├── manifest.json         # 拡張機能マニフェスト (Manifest V3)

│   └── handlers/

├── content/              # Content Script (ページDOM操作・再生位置取得)
│   ├── index.ts          # ライフサイクル管理、エントリポイント
│   ├── platforms/
│   │   ├── adapter.interface.ts # プラットフォーム共通インターフェース
│   │   ├── base.adapter.ts      # video要素共通操作の抽象クラス
│   │   └── twitch.adapter.ts    # Twitch固有のアダプター
│   ├── observers/
│   │   ├── chat.observer.ts     # チャット入力監視 (IME考慮 / 早期リターン)
│   │   └── command.observer.ts  # ショートカットキーコマンド監視
│   └── ui/
│       └── toast.manager.ts     # Shadow DOM トースト通知描画
├── popup/                # ポップアップUI (履歴確認・削除・設定)
│   ├── index.html
│   ├── index.css
│   └── index.ts          # Vanilla TSによる高速描画UI
└── common/               # 共有モジュール
    ├── models/           # 各種データモデル (Bookmark, Settings, Messages)
    ├── storage/          # StorageManager シングルトン
    └── utils/            # 汎用ヘルパー (時間パース、URL検証、サニタイズ)
```

## 🛠️ インストールとビルド手順

### 前提条件
- Node.js (v18以上推奨)
- npm

### 1. 依存パッケージのインストール
プロジェクトのルートディレクトリで以下を実行します。
```bash
npm install
```

### 2. 拡張機能のビルド
TypeScriptファイルをコンパイルし、静的アセット（HTML/CSS/マニフェスト）を `dist/` ディレクトリに出力します。
```bash
npm run build
```

### 3. Google Chromeへの導入方法
1. Chromeを開き、アドレスバーに `chrome://extensions/` を入力して拡張機能管理ページを開きます。
2. 右上の「**デベロッパー モード**」を有効にします。
3. 左上の「**パッケージ化されていない拡張機能を読み込む**」をクリックします。
4. 本プロジェクト配下にビルドされた `dist` ディレクトリを選択して読み込みます。

## 📖 使い方ガイド

### 1. キーボードショートカットで打刻 (MVP)
- Twitchの配信または動画（VOD）の視聴中に **`Alt + Shift + B`** を押します。
- 画面右上に「**ブックマークを保存中...**」から「**[時間] をブックマークしました**」へとトースト通知が表示されます。
- ショートカットキーは `chrome://extensions/shortcuts` からお好みのキーに変更可能です。

### 2. チャットで打刻
- チャット入力欄にデフォルトキーワード `!bm` を入力して送信します。
- 送信した瞬間を検知し、自動でその再生時間をブックマークに記録します。
- 日本語のIME入力中（漢字変換中）の Enter 押下では誤動作しないよう設計されています。

### 3. 履歴の確認と動画へのジャンプ
- Chromeのツールバーから拡張機能のアイコンをクリックしてポップアップを開きます。
- **履歴タブ**: 最新50件までの打刻履歴が表示されます。
  - 項目をクリックすると、安全なタイムスタンプ付きURL（例: `https://www.twitch.tv/videos/xxx?t=1h23m45s`）が自動生成され、新規タブで該当時間から再生が始まります。
  - ごみ箱アイコン `🗑` をクリックすると、その項目がUIおよびストレージから即時に消去されます。
- **設定タブ**:
  - 「**チャット監視を有効にする**」のトグルをOFFにすると、チャットのMutationObserverが無効化され、不要なバックグラウンド負荷が完全になくなります。
  - **キーワード管理**: `!bm` 以外のトリガーワード（例: `!clip` や `ここ！`）を自由に追加・削除できます。追加するとチャットでその言葉を打った際にも自動打刻されます。

## 📝 手動テスト仕様書（結合テスト用）

Chromeへ読み込ませた状態で、以下の手順に沿って手動テストを行うことができます。

| テスト項目 | 操作手順 | 期待される動作 |
|:---|:---|:---|
| **1. ショートカット打刻** | TwitchのVODページで `Alt+Shift+B` を押下。 | 右上にトーストが100ms以内に「保存中...」と表示され、その後「保存完了」に切り替わること。 |
| **2. フルスクリーン時のトースト表示** | VODをシアターモードまたはフルスクリーン表示にして `Alt+Shift+B` を押下。 | トーストが映像の最前面（映像の上に重なって）に正しく表示されること。 |
| **3. チャット打刻 (通常)** | チャット欄に `!bm` と入力して送信する。 | ショートカット時と同様にトーストが表示され、ポップアップに履歴が追加されること。 |
| **4. チャット打刻 (IME考慮)** | チャット欄で「こんにちは!bm」と入力し、変換確定のためにEnterを押す。 | 打刻がトリガーされないこと。（もう一度Enterを押して送信したタイミングでトリガーされること）。 |
| **5. チャット監視トグル動作** | 設定タブで「チャット監視を有効にする」をOFFにし、チャットで `!bm` を送信する。 | 打刻が一切動作しないこと。また、開発者ツールのPerformanceタブ等でMutationObserverの監視イベントが発生していないこと。 |
| **6. 安全な遷移確認** | ポップアップの履歴リストから項目をクリックする。 | 新しいタブで該当時間（例: `?t=1h02m30s`）から動画が再生されること。 |
| **7. 削除と即時反映** | 履歴から任意の項目の `🗑` ボタンをクリックする。 | 画面上のリストから該当項目がアニメーションとともに即時に消去されること。ポップアップを再起動しても消去されたままであること。 |
